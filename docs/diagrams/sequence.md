## SD-001: メッセージ送信フロー（正常系）

```mermaid
sequenceDiagram
    participant U as ユーザー
    participant FE as Frontend (Next.js)
    participant API as API Route (/api/chat)
    participant OAI as OpenAI API

    U->>FE: メッセージ入力 & 送信
    FE->>FE: ローディング状態ON
    FE->>FE: 会話履歴にユーザーメッセージ追加
    FE->>API: POST /api/chat { messages: [...] }
    API->>OAI: chat.completions.create(model: gpt-4o, messages)
    OAI-->>API: AIレスポンス
    API-->>FE: { role: "assistant", content: "..." }
    FE->>FE: 会話履歴にAIメッセージ追加
    FE->>FE: ローディング状態OFF
    FE-->>U: AIの返答を表示
```

- FR-004: ローディング・エラー表示

## SD-002: メッセージ送信フロー（エラー系）

```mermaid
sequenceDiagram
    participant U as ユーザー
    participant FE as Frontend (Next.js)
    participant API as API Route (/api/chat)
    participant OAI as OpenAI API

    U->>FE: メッセージ入力 & 送信
    FE->>FE: ローディング状態ON
    FE->>API: POST /api/chat { messages: [...] }
    API->>OAI: chat.completions.create(...)
    OAI-->>API: エラー（Rate limit / Network error）
    API-->>FE: 500 { error: "AI APIエラーが発生しました" }
    FE->>FE: ローディング状態OFF
    FE-->>U: エラーメッセージ表示
```
