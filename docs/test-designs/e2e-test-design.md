## E2E-001: 基本チャットフロー（ハッピーパス）

### シナリオ概要
ユーザーがメッセージを送信してAIの返答を受け取るまでの基本フロー

### テストステップ

| Step | 操作 | 期待結果 |
|------|------|----------|
| 1 | `http://localhost:3002` にアクセス | チャット画面が表示される |
| 2 | 空の状態を確認 | 「AIとの会話を始めましょう」が表示される |
| 3 | テキストエリアに「こんにちは」と入力 | 入力値が反映される |
| 4 | 送信ボタンをクリック | ユーザーメッセージが右側に表示される |
| 5 | ローディング確認 | ドットアニメーションが表示される |
| 6 | AIレスポンス受信 | AIメッセージが左側に表示される |
| 7 | ローディング消去確認 | アニメーションが消える |

### Playwrightテストコード

```typescript
import { test, expect } from '@playwright/test';

test('E2E-001: 基本チャットフロー', async ({ page }) => {
  await page.goto('http://localhost:3002');
  await expect(page.getByText('AIとの会話を始めましょう')).toBeVisible();

  await page.getByRole('textbox').fill('こんにちは');
  await page.getByRole('button', { name: '送信' }).click();

  // ユーザーメッセージ表示確認
  await expect(page.getByText('こんにちは')).toBeVisible();

  // ローディング表示確認
  await expect(page.locator('.animate-bounce').first()).toBeVisible();

  // AIレスポンス受信（最大15秒待機）
  await expect(page.locator('.rounded-bl-sm').first()).toBeVisible({ timeout: 15000 });

  // ローディング消去確認
  await expect(page.locator('.animate-bounce').first()).not.toBeVisible();
});
```

## E2E-002: Enterキー送信フロー

### テストステップ

| Step | 操作 | 期待結果 |
|------|------|----------|
| 1 | テキストエリアにテキスト入力 | 入力値が反映される |
| 2 | Enterキー押下 | メッセージが送信される |
| 3 | Shift+Enter押下 | 改行される（送信されない） |

```typescript
test('E2E-002: Enterキー送信', async ({ page }) => {
  await page.goto('http://localhost:3002');
  const textarea = page.getByRole('textbox');

  // Enterで送信
  await textarea.fill('Enterで送信テスト');
  await textarea.press('Enter');
  await expect(page.getByText('Enterで送信テスト')).toBeVisible();

  // Shift+Enterで改行（送信されない）
  await textarea.fill('1行目');
  await textarea.press('Shift+Enter');
  await textarea.type('2行目');
  await expect(textarea).toHaveValue('1行目\n2行目');
});
```

## E2E-003: 会話リセットフロー

### テストステップ

| Step | 操作 | 期待結果 |
|------|------|----------|
| 1 | メッセージを送受信する | 会話履歴が表示される |
| 2 | 「会話をリセット」ボタンをクリック | 会話履歴がクリアされる |
| 3 | 空の状態を確認 | 「AIとの会話を始めましょう」が表示される |
| 4 | リセットボタンが無効化確認 | ボタンが押せない状態 |

```typescript
test('E2E-003: 会話リセット', async ({ page }) => {
  await page.goto('http://localhost:3002');

  // リセットボタンが無効（初期状態）
  await expect(page.getByRole('button', { name: '会話をリセット' })).toBeDisabled();

  // メッセージ送信
  await page.getByRole('textbox').fill('テストメッセージ');
  await page.getByRole('button', { name: '送信' }).click();
  await expect(page.getByText('テストメッセージ')).toBeVisible();

  // リセットが有効になる
  await expect(page.getByRole('button', { name: '会話をリセット' })).toBeEnabled();

  // リセット実行
  await page.getByRole('button', { name: '会話をリセット' }).click();
  await expect(page.getByText('AIとの会話を始めましょう')).toBeVisible();
  await expect(page.getByText('テストメッセージ')).not.toBeVisible();
});
```
