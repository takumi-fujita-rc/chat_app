## UT-001: ChatInput コンポーネント

### テスト対象
`src/components/ChatInput.tsx`

### テストケース

| ID | 分類 | 操作 | 期待結果 |
|----|------|------|----------|
| UT-001-01 | 正常系 | テキスト入力後に送信ボタンクリック | `onSend` が入力テキストで呼ばれる |
| UT-001-02 | 正常系 | テキスト入力後にEnterキー押下 | `onSend` が呼ばれる |
| UT-001-03 | 正常系 | Shift+Enter押下 | 改行される（送信されない） |
| UT-001-04 | 正常系 | 送信後 | 入力フィールドが空になる |
| UT-001-05 | 異常系 | 空文字で送信ボタンクリック | `onSend` が呼ばれない |
| UT-001-06 | 異常系 | スペースのみで送信 | `onSend` が呼ばれない |
| UT-001-07 | 異常系 | `disabled=true` 時に送信 | `onSend` が呼ばれない |
| UT-001-08 | 境界値 | 日本語入力中（isComposing=true）にEnter | 送信されない（IME確定のみ） |

### テストコード例

```tsx
import { render, screen, fireEvent } from '@testing-library/react';
import ChatInput from '@/components/ChatInput';

test('UT-001-01: テキスト入力後に送信ボタンクリック', () => {
  const onSend = jest.fn();
  render(<ChatInput onSend={onSend} disabled={false} />);
  fireEvent.change(screen.getByRole('textbox'), { target: { value: 'こんにちは' } });
  fireEvent.click(screen.getByText('送信'));
  expect(onSend).toHaveBeenCalledWith('こんにちは');
});

test('UT-001-08: IME入力中のEnterで送信されない', () => {
  const onSend = jest.fn();
  render(<ChatInput onSend={onSend} disabled={false} />);
  const textarea = screen.getByRole('textbox');
  fireEvent.change(textarea, { target: { value: 'てすと' } });
  fireEvent.keyDown(textarea, { key: 'Enter', nativeEvent: { isComposing: true } });
  expect(onSend).not.toHaveBeenCalled();
});
```
